## Overview

A personal portfolio site presenting my projects, skills, and background. Built entirely with HTML and CSS — no JavaScript, no frameworks, no external UI libraries.

---

## Tech

- HTML5
- CSS3
- Google Fonts — Domine, Sniglet, Londrina Sketch

---

## Website Features

- **Decorative hero boxes** — Three white rotated squares sit in the hero section as layout decoration. The arrangement and angles were first designed in Canva, geometry was manually worked out from that sketch, then implemented in CSS using `position: relative` and `rotate`. Sizes and positions adapt at every breakpoint.
- **Fully responsive** — Layout adjusts across all screen sizes through four manually written `@media` breakpoints.
- **Accessibility** — Every image carries a descriptive `alt` attribute.
- **No external help** — Design, layout, code, and all decisions were made independently. No hints, references, or tutorials used at any stage.
